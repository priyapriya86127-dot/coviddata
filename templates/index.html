About

The project is built as a web experience focused on clarity and trust. It is designed to grow: you can plug in live data sources, educational content, or regional views as your requirements evolve.

Purpose

Surface reliable summaries and navigation so visitors quickly find what matters—overview, deeper detail, and next steps.

Audience

General public, students, and teams who need a lightweight portal without installing native apps.

Scope

Landing and information architecture today; ready to extend with dashboards, APIs, or authenticated areas later.

Technology used

The stack is intentionally small so you can run and deploy it easily on any machine or host that supports Python.

  • Python 3
  • Flask
  • Pandas
  • Jinja2
  • Chart.js
  • Plotly
  • HTML5 / CSS3
